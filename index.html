  • ferrograd [repo]

    • A small scalar autograd engine / Rust library, inspired from Karpathy's micrograd, with more features, such as implementations of extra activation functions, optimizers, loss criterions and accuracy metrics.

    Capable of MNIST classification. Example implemented using the library.

    Capable of creating neurons, dense layers and multilayer perceptrons (MLPs). Can be utilised for binary and multiclass/multilabel non-linear classification.

    • Loss criterions -
      Binary Cross-Entropy, Cross-Entropy, Hinge, MSE/RMSE.
    • Activation functions -
      ReLU, LeakyReLU, Sigmoid, tanh.

    Languages: rust

  • Kannada Handwriting Calculator [repo]

    • A calculator that uses handwritten Kannada digits and operators to calculate the result, using contour detection and ConvNet (Convolutional Neural Network) model prediction.

    • PyTorch is used to create, train and load the state of the neural network model used for predictions.
    • The CNN is trained on Kannada MNIST.
    • OpenCV and Pillow (PIL) are used to read input from the GUI canvas and to obtain contours for individual digits/operators.
    • CustomTKinter is used to provide the GUI.
    • The individual digits/operators are detected and their most probable target classes are predicted.
    • The predictions are combined into a string and evaluated to get the result.

    Languages: python
    Modules: pytorch, torchvision, opencv, numpy, pandas, customtkinter
    Tools: conda, jupyter notebooks

  • IMDb 250 Knowledge Graph RAG [repo]

    • Simple implementation of Graph RAG using IMDb Top 250, Neo4j Graph database and Gemini API.

    Neo4j database stores the network graph.

    • There are 3 types of nodes - Movie, Genre, Person Person can be - Actor, Director, Writer or a combination of these.
    • Relationships are -
      Person - [ WROTE / ACTED_IN / DIRECTED ] -> Movie
      Movie - [ BELONGS_TO ] -> Genre

    LangChain and Gemini are used for the pipeline, which

    • Processes the natural language prompt
    • Generates a Cypher query
    • Queries the Neo4j database with generated query and gets back the result in JSON Parses the JSON and responds back in natural language

    Languages: python
    Modules: langchain, neo4j-python, numpy, polars
    Tools: neo4j, conda, jupyter notebooks
